# 草药学

添加6种草药，可以用来制作工具、药水、合成装置、附魔装置、刷怪装置、物流装置。

## 方块

### 木头

- [x] **红樱木原木**：可以合成红樱木木板、用红樱木木板合成红樱木木棍。
  - [x] **红樱木树苗**：红樱木的树苗。
  - [x] **红樱木木板**：用来制作本模组的合成和生产装置。
  - [x] **红樱木木棍**：用来制作本模组的工具和武器。
    - [ ] **魔法木棍**：升级版的红樱木棍
    - [ ] **繁花木棍**：比魔法木棍更高级的升级版的红樱木棍
  - [x] **红樱木树叶**：破坏或者自然凋零后概率掉落红樱木树苗和森林浆果，底部可以用于种植森林浆果。
  - [x] **红樱桃果**：可以种植，可以用来吃，可以改良。
    - [ ] **精灵果**：由精灵果植物收获得到。
    - [ ] **精灵果植物**：使用金铃花右键森林浆果植物可以令其变成1阶段的精灵果植物。

### 石头

- [x] **微光石**：类似于红樱木，是本模组的石头变体。发出亮度7的微弱光芒。
  - [x] **微光石砖**：微光石的石砖形态。发出亮度7的微弱光芒。
    - [x] **符文石砖**：上有符文的微光石砖。发出亮度15的强光（和萤石一样）。
    - [x] **微光石砖台阶**：微光石砖的台阶状态。发出亮度7的微弱光芒。
    - [x] **微光石砖楼梯**：微光石砖的楼梯状态。发出亮度7的微弱光芒。

### 沙土

- [ ] **草药种植土**：不需要耕即可种植草药，除此之外没有其他特殊之处。
  - [ ] **肥沃草药种植土**：使种植在上面的草药获得2倍的收成。
  - [ ] **(形式未确定)草药种植土**：使种植在上面的草药获得2倍的生长速度。
  - [ ] **(形式未确定)草药种植土**：非常昂贵，使种植在上面的草药获得4倍的增长速度且拥有4倍的收成。

  - [ ] **治愈花**：用来制作绒甲
    - [x] **绒甲**：使用治愈花和羽毛制作，用来制作花韵套的护甲
- [ ] **焦藤**：下界的怪物低概率掉落，用来制作坚韧藤蔓
  - [ ] **坚韧藤蔓**：用来制作丝甲
    - [x] **丝甲**：使用坚韧藤蔓和羊毛制作，用来制作深林套的护甲
- [ ] **失忆残响**：末地的怪物低概率掉落，用来制作**记忆之心**
  - [ ] **记忆之心**：用来给深林套和璀璨宝石工具升级，和深林套或璀璨宝石工具在桌台上锻造之后可以使二者的掉落物拥有灵魂绑定的特性以及不会被熔岩、仙人掌、爆炸、火焰毁坏的特性，且丢出去后会被快速吸入
- [ ] **花束**：需要每种草药产物各1个，用来制作后期产物

## 草药

### 主世界草药

- [x] **绿鳞蕨**：生成在主世界的植物，会发出微弱的光。
  - [x] **绿鳞蕨种子**：可以用来种植绿鳞蕨作物。
  - [x] **绿鳞蕨作物**：有8个生长阶段，成熟后可以收获1个蕨鳞和1个绿鳞蕨种子。
  - [x] **鳞甲(Scaleplate)**：绿鳞蕨的叶子，非常坚硬且蕴含着生命的脉动，用于制作装备、工具，特点是这种草药参与制作的工具和装备可以自动恢复耐久。

- [x] **雨露花**：生成在主世界的植物，会发出微弱的光。
  - [x] **雨露花籽**：可以用来种植雨露花作物。
  - [x] **雨露花作物**：有8个生长阶段，成熟后可以收获1个灵瓣和1个雨露花籽。
  - [x] **雨露花瓣(Dewpetal Shard)**：雨露花的花瓣，对水的亲和力很高，用于制作药水相关的物品。

- [x] **和风铃兰**：生成在主世界的植物，会发出微弱的光。
  - [x] **和风铃兰花籽**：可以用来种植和风铃兰作物。
  - [x] **和风铃兰作物**：有8个生长阶段，成熟后可以收获1个金铃花和1个和风铃兰花籽。
  - [x] **金铃花(Golden Lilybell)**：和风铃兰的花朵，很受Sylvana喜爱，用于制作包括精灵契约的物流相关的物品。

### 下界草药

- [x] **晶芽**：生成在下界的植物，会发出微弱的光以及晶化特效。
  - [x] **晶化种子(Crystalization Seed)**：可以用来种植晶芽作物。
  - [x] **晶芽作物**：有8个生长阶段，成熟后可以收获1个晶核和1个晶化种子。
  - [x] **晶刺(Cryst Spine)**：晶芽的叶子，由于生长在大地的深处，叶片硬的像刺，用于制作生产设备。

- [x] **炼狱草**：生成在下界的植物，会发出微弱的光以及火焰特效。
  - [x] **能量种子(Emberseed)**：可以用来种植炼狱草作物。
  - [x] **炼狱草作物**：有8个生长阶段，成熟后可以收获1个燃烧草段和1个能量种子。
  - [x] **燃烧草段(Burnt Node)**：用于升级原版设备。

### 末地草药

- [x] **星语蔷薇**：生成在末地的植物，会发出微弱的光以及末影星空特效。
  - [x] **神秘种子(Mysterious Seed)**：可以用来种植星语蔷薇作物。
  - [x] **星语蔷薇作物**：有8个生长阶段，成熟后可以收获1个星忆花心和1个异域之种。
  - [x] **星忆花心(Heart of Stardream)**：用于制作更厉害的设备。

## 工具

- [ ] **织流环(Flowweave Ring)**
  - **调和**：相当于这个模组的扳手，可以用来施法，启动合成，启动炼药，启动附魔，给Sylvana进行绑定等等。
  - **施法**：右键有药水的釜可以将其浸泡在药水内，过一段时间织流环会完全吸收釜里面的药水，药水亲和力会大大提升，此时对着右键不会触发织流环的特殊事件的方块右键时会进行施法，施法后会消耗背包或便携式箱子里面的酿造这个药水时所需的草药，如果没有足够草药，则施法不会成功。施法有三个模式（对着空气或者不会触发效果的方块shift右键可以切换模式，默认为灌注模式）：
    - **灌注模式**：使自己获得对应药水的buff
    - **震荡模式**：发射一道波对命中的地方发生爆炸并使爆炸范围内的生物获得buff，这个爆炸只会发出声音和造成特效，不会有爆炸的效果，爆炸范围为TNT爆炸范围，该波最远飞到64格，该模式施法需要消耗原本草药消耗数量的1.5倍的草药（向下取整）
    - **滞留模式**：发射一道波对命中的地方发生爆炸并使爆炸范围内生成对应buff的滞留药水云，这个爆炸只会发出声音和造成特效，不会有爆炸的效果，爆炸范围为TNT爆炸范围，该波最远飞到64格，该模式施法需要消耗原本草药消耗数量的2倍的草药（向下取整）
- [ ] **草药师手记本**
  - [ ] **主人**：谁做出来的，主人就是谁。
  - [ ] **笔记**：主界面，相当于这个模组的指南，里面可以看到每个模块的物品的制作方式、用处等等。
  - [ ] **日记**：不是真正的页面，而是以书签的形式出现在左侧，书签上会显示日记的标题和目前的章节，每多写一个日记就多一个书签。点击某个书签后这个书签会展开，然后显示**遗忘**和**撕毁**两个按钮。
    - [ ] **遗忘**：在书签上把这个日记标题的字加上横线，并且在物品栏悬停时的tooltip上也加横线，日记提供的buff临时消失。点击之后，这个按钮会变成**回忆**，点击回忆后会恢复原来的状态。
    - [ ] **撕毁**：删掉这个种类的日记，并且在tooltip上也不再显示这个日记，日记提供的buff永久消失。
    - [ ] **新篇章**：如果有了某种日记的某一章，可以在桌台上用另外的物品给草药师手记本添加该日记的下一章。也就是如果我解锁了旅行日记的第一章，那么我才能写入第二章，如果没有，则用第二章所需的材料制作会无法制作。新的篇章可以增强日记的buff。
  - [ ] **珍贵的记忆**：草药师手记本的包装形式，只有主人可以将自己的草药师手记本做成珍贵的回忆，给别人后，别人右键使用，可以将里面的草药师手记本的主人变成自己。
  - [ ] **联动**：如果有饰品栏模组时，可以携带在任何部位，如果携带在饰品栏上，按C键可以打开。
- [ ] **扫帚**
  - [ ] **飞行**：无论是否拿在手上，放在背包里的时候，可以双击空格飞行，此时和创造模式飞行一样，但是人物会显示坐着的样子，且坐在扫帚上。
  - [ ] **扫地**：拿在手上时，右键方块，会处理附近一定范围内的掉落物，暂定是以右键的方块为中心5\*5\*5，且会显示焕然一新的粒子特效（白色红石特效）。有两个模式，可以通过shift右键空气和非交互方块（和织流环一样）切换模式：
    - **清扫模式**：将附近掉落物清除。
    - **回收模式**：将附近掉落物装入背包。
- [x] **草药箱**
  - [x] **收集**：拿在手上的时候，右键将背包里所有草药放进去，shift右键将箱子里所有草药拿出来，拿出来的时候只会填满背包，如果背包装不下则剩在草药箱里面。
  - [x] **拿取**：如果右键草药柜方块，则不会触发上述逻辑，而是会从草药柜里取出足够填满草药箱的草药填满草药箱，如果某种草药不够，就取出全部，shift右键则是用草药箱内的草药将草药柜填满。每种草药可以存放512个。
  - [x] **背负**：除此之外，放在物品栏里的时候会在背上渲染这个模型，相当于背着这个箱子，拿在手上的时候则会显示在手上，而不是背上。如果穿了胸甲，则渲染的时候往外渲染1格，脱掉之后恢复原位。

- [ ] **花韵套(Florawyn Armor)**：中级护甲，中期用的护甲，0.5秒恢复1点耐久。有1个花韵套部件时，20秒恢复1点生命，有2个花韵套部件时，15秒恢复1点生命，有3个花韵套部件时，10秒恢复1点生命，有4个花韵套部件时，5秒恢复1点生命。

## 生产

- [ ] **强化合金砧**：会自动修复的铁砧，且降低附魔时的消耗和增长，并使附魔和修理不会出现过于昂贵
- [ ] **水坛**：用装有至少8次饮用次数的水壶右击，会消耗8次饮用次数并将水坛装满这种药水，此时水坛内将会渲染出该药水颜色的水。水坛有1个槽，只能用来装雨露花瓣。当水坛里既有药水又有雨露花瓣时，会使附近的玩家和生物获得药水对应的buff，由于药水通过扩散传播，因此无论原药水等级是多少，扩散出来的药水等级都是1，每1秒上1次buff，每60秒消耗1个雨露花瓣，工作时渲染出对应药水颜色的粒子特效向上飘。除了装药水以外，还能用来装普通的水，装普通水的时候，可以自动填充附近的釜的水，釜空的时候，会出现一个2秒的粒子特效并在2秒后加满水，该过程不需要消耗雨露花，工作时渲染出水颜色的粒子特效向上飘。使用织流环shift右键可以清空药水与水。
- [ ] **盆栽花盆**：用于栽培晶体植株。没有gui，但是有8个抽象的槽，通过右键交互，一个用来装填土类，只能放1个物品；一个用来装填幼苗，也只能放1个物品；另外6个用来装填草药，可以放每种草药1组。用土类右键可以将土类放进土类槽，此时盆栽里会渲染出对应土类的样子，可以定义可以放什么土类，什么即使不是真的土类；用幼苗右键可以将幼苗放进幼苗槽，此时盆栽上1格会渲染一个缩小到1格的对应幼苗的材质；用草药右键可以将草药放进草药槽，此时对准盆栽会像草药柜一样渲染出当前盆栽内有的草药的种类和数量，不过盆栽外观不会发生改变。草药槽的草药足够后，植株会消耗所需的草药并开始生长，生长时间较为漫长，生长完后产物会直接掉落到地上。幼苗不会被消耗掉，只会消耗草药。因此一个合成配方需要定义：植株幼苗，适配土壤，渲染模型，产物，所需草药
  - [ ] **晶体植株**：可以使用不同数量的草药培育出不同的矿石。每种植株需要一个种子，种子的获取难度较高，可能要抽卡或者打怪，还没确定。植株在盆栽丄渲染的幼苗材质为一个橡树的形状，但是树叶的颜色变成对应矿石的颜色。
