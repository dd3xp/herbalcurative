# 草药学

添加5种草药，可以用来制作工具、药水、合成装置、附魔装置、刷怪装置、物流装置。

## 方块

### 木头

- [x] **红樱木原木**：可以合成红樱木木板、用红樱木木板合成红樱木木棍。
  - [x] **红樱木树苗**：红樱木的树苗。
  - [x] **红樱木木板**：用来制作本模组的合成和生产装置。
  - [x] **红樱木木棍**：用来制作本模组的工具和武器。
  - [x] **红樱木树叶**：破坏或者自然凋零后概率掉落红樱木树苗和森林浆果，底部可以用于种植森林浆果。
  - [x] **森林浆果**：可以种植，可以用来吃，可以改良。
    - [ ] **精灵果**：由精灵果植物收获得到。
    - [ ] **精灵果植物**：使用金铃花右键森林浆果植物可以令其变成1阶段的精灵果植物。

### 石头

- [ ] **微光石**：类似于红樱木，是本模组的石头变体。
  - [ ] **微光石砖**：微光石的石砖形态。
    - [ ] **符文石砖**：上有符文的微光石砖。
    - [ ] **微光石砖台阶**：微光石砖的台阶状态。

### 沙土

- [ ] **草药种植土**：不需要耕即可种植草药，除此之外没有其他特殊之处。
  - [ ] **肥沃草药种植土**：使种植在上面的草药获得2倍的收成。
  - [ ] **(形式未确定)草药种植土**：使种植在上面的草药获得2倍的生长速度。
  - [ ] **(形式未确定)草药种植土**：非常昂贵，使种植在上面的草药获得4倍的增长速度且拥有4倍的收成。

## 草药

### 主世界草药

- [x] **绿鳞蕨**：生成在主世界的植物，会发出微弱的光。
  - [x] **绿鳞蕨种子**：可以用来种植绿鳞蕨作物。
  - [x] **绿鳞蕨作物**：有8个生长阶段，成熟后可以收获1个蕨鳞和1个绿鳞蕨种子。
  - [x] **鳞甲(Scaleplate)**：绿鳞蕨的叶子，非常坚硬且蕴含着生命的脉动，用于制作装备、工具，特点是这种草药参与制作的工具和装备可以自动恢复耐久。

- [x] **雨露花**：生成在主世界的植物，会发出微弱的光。
  - [x] **雨露花籽**：可以用来种植雨露花作物。
  - [x] **雨露花作物**：有8个生长阶段，成熟后可以收获1个灵瓣和1个雨露花籽。
  - [x] **雨露花瓣(Dewpetal Shard)**：雨露花的花瓣，对水的亲和力很高，用于制作药水相关的物品。

- [x] **和风铃兰**：生成在主世界的植物，会发出微弱的光。
  - [x] **和风铃兰花籽**：可以用来种植和风铃兰作物。
  - [x] **和风铃兰作物**：有8个生长阶段，成熟后可以收获1个金铃花和1个和风铃兰花籽。
  - [x] **金铃花(Golden Lilybell)**：和风铃兰的花朵，很受Sylvana喜爱，用于制作包括精灵契约的物流相关的物品。

### 下界草药

- [x] **晶芽**：生成在下界的植物，会发出微弱的光以及晶化特效。
  - [x] **晶化种子(Crystalization Seed)**：可以用来种植晶芽作物。
  - [x] **晶芽作物**：有8个生长阶段，成熟后可以收获1个晶核和1个晶化种子。
  - [x] **晶刺(Cryst Spine)**：晶芽的叶子，由于生长在大地的深处，叶片硬的像刺，用于制作生产设备。

- [x] **炼狱草**：生成在下界的植物，会发出微弱的光以及火焰特效。
  - [x] **能量种子(Emberseed)**：可以用来种植炼狱草作物。
  - [x] **炼狱草作物**：有8个生长阶段，成熟后可以收获1个焰核和1个能量种子。
  - [x] **焰节(Burnt Node)**：用于升级原版设备。

### 末地草药

- [x] **星语蔷薇**：生成在末地的植物，会发出微弱的光以及末影星空特效。
  - [x] **神秘种子(Mysterious Seed)**：可以用来种植星语蔷薇作物。
  - [x] **星语蔷薇作物**：有8个生长阶段，成熟后可以收获1个星忆花心和1个异域之种。
  - [x] **星忆花心(Heart of Stardream)**：用于制作更厉害的设备。

## 工具

- [ ] **织流环(Flowweave Ring)**
  - **调和模式**：相当于这个模组的扳手，可以用来施法，启动合成，启动炼药，启动附魔，给Sylvana进行绑定等等。shift右键切换施法状态和工作状态，合成出来后默认为工作状态。
  - **施法模式**：右键有4B药水的釜可以将其浸泡在药水内，过一段时间织流环会完全吸收釜里面的药水，且的药水亲和力会大大提升，此时对着右键不会触发织流环的特殊事件的方块右键时会进行施法，施法后会获得对应药水的buff且背包或便携式箱子里面的酿造对应药水1单位所需的草药，如果没有足够草药，则施法不会成功。
- [ ] **草药师手记本**
  - [ ] **主人**：谁做出来的，主人就是谁。
  - [ ] **笔记**：相当于这个模组的指南，里面可以看到每个模块的物品的制作方式、用处等等。分为6个模块，对应每个草药的职责。
  - [ ] **日记**：可以看到所有日记的页面，每多写入一种日记就多一个页面，并且在tooltip上会显示这个日记的种类，日记可以提供不同buff，但是只有在这个草药师笔记本的主人包里才会提供。右边有编辑、遗忘和撕毁三个选择。
    - [ ] **编辑**：可以编辑日记内的文字，有一个编辑字数上限。
    - [ ] **遗忘**：把这个页面的字变成加横线的字，并且在tooltip上也加横线，日记提供的buff临时消失。点击之后，这个按钮会变成**回忆**，点击回忆后会恢复原来的状态。
    - [ ] **撕毁**：删掉这个种类的日记，并且在tooltip上也不再显示这个日记，日记提供的buff永久消失。
    - [ ] **新篇章**：一个种类的日记会有新的篇章，新的篇章可以增强日记的buff。
  - [ ] **珍贵的回忆**：草药师手记本的包装形式，只有主人可以将自己的草药师手记本做成珍贵的回忆，给别人后，别人右键使用，可以将里面的草药师手记本的主人变成自己。
- [ ] **扫帚**
  - [ ] **飞行**：无论是否拿在手上，放在背包里的时候，可以双击空格飞行，此时和创造模式飞行一样，但是人物会显示坐着的样子，且坐在扫帚上。
  - [ ] **扫地**：拿在手上时，右键方块，会清理附近一定范围内的掉落物，暂定是以右键的方块为中心5\*5\*5，且会显示焕然一新的粒子特效（白色红石特效）。
- [x] **草药箱**
  - [x] **收集**：拿在手上的时候，右键将背包里所有草药放进去，shift右键将箱子里所有草药拿出来，拿出来的时候只会填满背包，如果背包装不下则剩在草药箱里面。
  - [x] **拿取**：如果右键草药柜方块，则不会触发上述逻辑，而是会从草药柜里取出足够填满草药箱的草药填满草药箱，如果某种草药不够，就取出全部，shift右键则是用草药箱内的草药将草药柜填满。每种草药可以存放512个。
  - [x] **背负**：除此之外，放在物品栏里的时候会在背上渲染这个模型，相当于背着这个箱子，拿在手上的时候则会显示在手上，而不是背上。如果穿了胸甲，则渲染的时候往外渲染1格，脱掉之后恢复原位。

## 合成

### 制作台

- [ ] **制作台**：用左边的工具把中间的物品用右边的材料加工成成品形式。左边放一堆工具，中间放待加工的物品，右边放材料，左边3个格子，中间1个格子，右边6个格子，每个格子可以放堆叠上限个物品。用织流环右键中间可以消耗掉左边所需工具的1点耐久、中间和右边的对应数量东西制作1个成品，shift右键则是将最少的材料消耗完并制作对应数量的物品，制作好之后直接从中间弹出来。只有右边的材料种类严格符合配方才能制作，多了或者少了就不能，数量则无所谓。支持返还。
  - [ ] **雕刻凿**：可以附魔耐久。
    - [ ] **精灵之唤**：物品形态的精灵，用来叫精灵来帮忙，右键某个方块后放下并将精灵的工作区绑定为这个方块。
      - [ ] **流浪者**：默认职业，什么都不会干，只会站在原地。
      - [ ] **搬运工**：将工作区的物品搬运到目标方块，可以配置黑名单和白名单。手持小箱子。
      - [ ] **补给工**：将目标方块的物品放入工作区，可以配置黑名单和白名单。手持小箱子。
      - [ ] **园艺师**：当工作区为树叶，在树叶旁边裁剪，以缓慢的速度产生树叶和原木（没有树苗，因为后续会制作树叶转换树苗的方法配方），将产物放入目标方块。手持园艺大剪刀。
      - [ ] **采摘工**：当工作区为容器，将检查附近一定范围内的作物成熟情况，如果有成熟作物，飞到成熟作物处，将成熟作物返回第一阶段且收获对应数量的成熟作物的收获物。手持剪刀。

### 草药调和架(多方块结构)

- [x] **花篮**：可以装某种草药256个。空白的花篮可以通过拿着每种右键放入任意一种草药进行绑定，放入后，这个花篮只能放入这种草药，拿着其他草药右键就放入不了了，空手右键则可以将里面的草药拿出来，一次拿一组，不足一组有多少拿多少。和草药柜一样，对准花篮可以显示里面对应草药的材质和数量，如果没有，则会显示绑定的草药和0，没有绑定任何草药则什么都不显示。打掉之后会掉落里面的草药。右键放入草药，右键双击时，如果已经绑定了某个草药，则会像右键双击草药柜一样，把所有这种草药放进去，直到放满。空手shift右键取出。和火把一样，放在地上有放在地上的材质，挂在墙上有挂在墙上的材质

  - [ ] **晶体植株**：可以使用不同数量的草药培育出不同的矿石。每种植株需要一个种子，种子的获取难度较高，可能要抽卡或者打怪，还没确定。

    - [ ] **红石植株**：4晶芽，4星忆花心
    - [ ] **萤石粉植株**：4和风铃兰，4晶芽，4焰节
    - [ ] **下界石英植株**：4鳞甲，4晶芽，4焰节
    - [ ] **金锭植株**：8鳞甲，8和风铃兰
    - [ ] **青金石植株**：8雨露花瓣，8晶芽
    - [ ] **钻石植株**：8雨露花瓣，8晶芽，8星忆花心
    - [ ] **绿宝石植株**：8鳞甲，8晶芽，8星忆花心
