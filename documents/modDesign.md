# 草药学

添加5种草药，可以用来制作工具、药水、合成装置、附魔装置、刷怪装置、物流装置。

## 材料

### 红樱木

- [x] **红樱木树苗**：红樱木的树苗。
- [x] **红樱木原木**：可以合成红樱木木板、用红樱木木板合成红樱木木棍。
- [x] **红樱木木板**：用来制作本模组的合成和生产装置。
- [x] **红樱木木棍**：用来制作本模组的工具和武器。
  - [ ] **软甲**：用来制作最低级的装备。
- [x] **红樱木树叶**：破坏或者自然凋零后概率掉落红樱木树苗和森林浆果，底部可以用于种植森林浆果。
- [x] **森林浆果**：可以种植，可以用来吃，可以改良。
  - [ ] **精灵果**：由精灵果植物收获得到。
  - [ ] **精灵果植物**：使用金铃花右键森林浆果植物可以令其变成1阶段的精灵果植物。

### 蕴魔石头

- [ ] **蕴魔石头**：类似于红樱木，是本模组的石头变体。
  - [ ] **蕴魔石砖**：蕴魔石头的石砖形态。
    - [ ] **符文石砖**：上有符文的蕴魔石砖。
    - [ ] **蕴魔石砖台阶**：蕴魔石砖的台阶状态。

### 草药种植土

- [ ] **草药种植土**：不需要耕即可种植草药，除此之外没有其他特殊之处。
- [ ] **肥沃草药种植土**：使种植在上面的草药获得2倍的收成。
- [ ] **(形式未确定)草药种植土**：使种植在上面的草药获得2倍的生长速度。
- [ ] **(形式未确定)草药种植土**：非常昂贵，使种植在上面的草药获得4倍的增长速度且拥有4倍的收成。

## 草药

### 主世界草药

#### 绿鳞蕨(Verdscale Fern)

- [x] **绿鳞蕨**：生成在主世界的植物，会发出微弱的光。
- [x] **绿鳞蕨种子**：可以用来种植绿鳞蕨作物。
- [x] **绿鳞蕨作物**：有8个生长阶段，成熟后可以收获1个蕨鳞和1个绿鳞蕨种子。
- [x] **鳞甲(Scaleplate)**：绿鳞蕨的叶子，非常坚硬且蕴含着生命的脉动，用于制作装备、工具，特点是这种草药参与制作的工具和装备可以自动恢复耐久。

#### 雨露花(Dewpetal)

- [x] **雨露花**：生成在主世界的植物，会发出微弱的光。
- [x] **雨露花籽**：可以用来种植雨露花作物。
- [x] **雨露花作物**：有8个生长阶段，成熟后可以收获1个灵瓣和1个雨露花籽。
- [x] **雨露花瓣(Dewpetal Shard)**：雨露花的花瓣，对水的亲和力很高，用于制作药水相关的物品。

#### 和风铃兰(Zephyr Lily)

- [x] **和风铃兰**：生成在主世界的植物，会发出微弱的光。
- [x] **和风铃兰花籽**：可以用来种植和风铃兰作物。
- [x] **和风铃兰作物**：有8个生长阶段，成熟后可以收获1个金铃花和1个和风铃兰花籽。
- [x] **金铃花(Golden Lilybell)**：和风铃兰的花朵，很受Sylvana喜爱，用于制作包括精灵契约的物流相关的物品。

### 下界草药

#### 晶芽(Crystbud)

- [x] **晶芽**：生成在下界的植物，会发出微弱的光以及晶化特效。
- [x] **晶化种子(Crystalization Seed)**：可以用来种植晶芽作物。
- [x] **晶芽作物**：有8个生长阶段，成熟后可以收获1个晶核和1个晶化种子。
- [x] **晶刺(Cryst Spine)**：晶芽的叶子，由于生长在大地的深处，叶片硬的像刺，用于制作生产设备。

#### 炼狱草(Pyrisage)

- [x] **炼狱草**：生成在下界的植物，会发出微弱的光以及火焰特效。
- [x] **能量种子(Emberseed)**：可以用来种植炼狱草作物。
- [x] **炼狱草作物**：有8个生长阶段，成熟后可以收获1个焰核和1个能量种子。
- [x] **焰节(Burnt Node)**：用于升级原版设备。

### 末地草药

#### 星语蔷薇(Rosynia)

- [x] **星语蔷薇**：生成在末地的植物，会发出微弱的光以及末影星空特效。
- [x] **神秘种子(Mysterious Seed)**：可以用来种植星语蔷薇作物。
- [x] **星语蔷薇作物**：有8个生长阶段，成熟后可以收获1个星忆花心和1个异域之种。
- [x] **星忆花心(Heart of Stardream)**：用于制作更厉害的设备。

## 工具

- [ ] **织流环(Flowweave Ring)**
  - 相当于这个模组的扳手，可以用来施法，启动合成，启动炼药，启动附魔，给Sylvana进行绑定等等。shift右键切换施法状态和工作状态，合成出来后默认为工作状态。
  - 右键有4B药水的釜可以将其浸泡在药水内，过一段时间织流环会完全吸收釜里面的药水，且的药水亲和力会大大提升，此时对着右键不会触发织流环的特殊事件的方块右键时会进行施法，施法后会获得对应药水的buff且背包或便携式箱子里面的酿造对应药水1单位所需的草药，如果没有足够草药，则施法不会成功。
- [ ] **扫帚**
  - 很昂贵的工具，需要下界之星等昂贵的材料制作。可以用来飞行和扫地。
  - 放在背包里的时候背上会显示背着这个扫帚，拿在手上则会显示在手上，而不是背上。
  - 无论是否拿在手上，放在背包里的时候，可以双击空格飞行，此时和创造模式飞行一样，但是人物会显示坐着的样子，且坐在扫帚上。
  - 拿在手上时，右键方块，会清理附近一定范围内的掉落物，暂定是以右键的方块为中心5\*5\*5，且会显示焕然一新的粒子特效（白色红石特效）。
- [ ] **草药箱**
  - 拿在手上的时候，右键将背包里所有草药放进去，shift右键将箱子里所有草药拿出来，拿出来的时候只会填满背包，如果背包装不下则剩在草药箱里面。
  - 如果右键草药柜方块，则不会触发上述逻辑，而是会从草药柜里取出足够填满草药箱的草药填满草药箱，如果某种草药不够，就取出全部，shift右键则是用草药箱内的草药将草药柜填满。每种草药可以存放1024个。
  - 除此之外，放在物品栏里的时候会在背上渲染这个模型，相当于背着这个箱子，拿在手上的时候则会显示在手上，而不是背上。如果穿了胸甲，则渲染的时候往外渲染1格，脱掉之后恢复原位。

### 草药调和架(多方块结构)

- [ ] **花篮**：可以装某种草药256个，但是需要和对应草药在工作台合成绑定是装什么草药的。空白的花篮也可以通过右键放入任意一种草药进行绑定。和另一草药合成则合成的时候会转换成另一个草药的花篮，绑定和转换的合成不会消耗草药。打掉之后会掉落里面的草药。右键放入草药，空手右键取出。
- [ ] **红樱木架子**：可以将物品放在架子上，但不是晒干架那种架子，而是类似于一个物品展示框那种方形的架子。只能放1个物品，需要依附在某个面上。右键放入物品，有物品时，右键取出。
- [ ] **草药调和架(多方块结构)**：
    当玩家对红樱木架子右键使用织流环时，会检测以面向方向为前方的上、下、左、右、左上、右上、左下、右下是否有红樱木架子，同时还会检测左边的第二格、左边的第二个的上面一格和下面一格，以及右边的第二格、右边的第二格的上面一格和下面一格，总共6格是否有花篮。如果全部满足，则满足多方块结构。多方块结构应如下所示（这是一个在墙上的示意图，不是放地上的）：

## 生物

### 友好生物

- [ ] **精灵(Sylvana)**：会在成熟的精灵果下面第一个合法的方块上生成，生成后，对应的精灵果植物方块会消失。可以用织流环左键变成物品形态的精灵，打死什么都不掉。职业为默认职业流浪者。每个职业都可以配置黑名单和白名单，默认黑名单模式。黑名单模式和白名单模式的时候身上穿的衣服会改变。
  - [ ] **精灵之唤**：物品形态的精灵，用来叫精灵来帮忙，右键某个方块后放下并将精灵的工作区绑定为这个方块。
    - [ ] **流浪者**：默认职业，什么都不会干，只会站在原地。
    - [ ] **搬运工**：将工作区的物品搬运到目标方块，可以配置黑名单和白名单。手持小箱子。
    - [ ] **补给工**：将目标方块的物品放入工作区，可以配置黑名单和白名单。手持小箱子。
    - [ ] **园艺师**：当工作区为树叶，在树叶旁边裁剪，以缓慢的速度产生树叶和原木（没有树苗，因为后续会制作树叶转换树苗的方法配方），将产物放入目标方块。手持园艺大剪刀。
    - [ ] **采摘工**：当工作区为容器，将检查附近一定范围内的作物成熟情况，如果有成熟作物，飞到成熟作物处，将成熟作物返回第一阶段且收获对应数量的成熟作物的收获物。手持剪刀。
    - [ ] **调香师**：当工作区为调香盘，将从目标方块内拿取所需草药进行调香，可以配置要调什么香料。手持花槽。
    - [ ] **栽培师**：当工作区为盆栽，将从目标方块内拿去所需草药进行植株栽培，可以配置要培育什么矿石的植株，手持花槽。

### 敌对生物

- [ ] **(小怪名称未确定)**：小怪，会掉落六种草药，但是每种概率都很低。
- [ ] **(Boss名称未确定)**：Boss，会掉落多种重要的后期材料，手持万象环，因此死后也会掉落万象环。
  - [ ] **万象环**：和织流环类似，但是拿在手上的时候有8点攻击力，且施法后得到的持续时间*1.5。
